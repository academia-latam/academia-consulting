## Structure to avoid (‚Üí fix)
1. Rule of three (forced triads) ‚Üí use 1, 2, or 4. 2. "Not just X but Y" / "It's not about X ‚Äî it's about Y" ‚Üí state it directly. 3. ## Human moves
Specificity over abstraction ¬∑ real numbers (never fabricated) ¬∑ name real things (Postgres, March 2026 report) ¬∑ concrete nouns + plain verbs ¬∑ varied rhythm, one idea per line ¬∑ a real opinion ¬∑ contractions; start a sentence with And/But ¬∑ occasional fragment ¬∑ read aloud ("would I say this to a colleague?"); Orwell: cut every word you can.

## Final review pass (procedure)
1. Grep banned list (case-insensitive). 2. Count em-dashes/exclamations (~0). 3. Triad sweep ("X, Y, and Z" adjective lists). 4. Rhythm check (3+ same-length sentences). 5. "Not just X but Y" sweep. 6. Read every sentence aloud. 7. Kill question: "Could this appear on a competitor's site?" If yes, add a fact only we can claim.
